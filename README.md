Импорт файла:<br>
Текстовый файл с расширением .txt. Структура:<br>
1. Первая строка – одно целое число N – количество вершин.<br>
2. Вторая строка – одно целое число M – количество рёбер.<br>
3. Следующие M строк описывают рёбра: u v w, где u, v – номера вершин (начиная с 0 или 1, указывается), w – вес ребра (целое или вещественное > 0).<br>
<br>
Редактирование графа:<br>
- ЛКМ по пустому месту — добавить вершину<br>
- Перетаскивание вершины — переместить<br>
- Ctrl+клик по двум вершинам — создать ребро<br>
- Shift+перетаскивание между вершинами — тоже создать ребро<br>
- ПКМ по вершине/ребру — удалить (контекстное меню)<br>
- Delete/Backspace — удалить выделенное<br>
- Alt+перетаскивание фона — панорамирование<br>
- Колесо мыши — зум
